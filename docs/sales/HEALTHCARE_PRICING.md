## Why This Platform Commands Premium Pricing

### Static Tools (Mineral, ThinkHR, SHRM) — $3–8 PEPM
- Pre-written compliance content library
- Human reads content, makes their own decisions
- No case management, no analysis, no workflows
- Content updated quarterly at best

### This Platform — Dynamic/Agentic
Every core module runs AI agent workflows that actively do work:

| Module | What the AI Agent Actually Does |
|---|---|
| **Compliance Engine** | Actively researches jurisdiction requirements via Gemini, streams real-time scan results, detects wage violations across locations, auto-fills missing compliance data |
| **IR Incidents** | AI categorizes incidents by type/severity, performs root cause analysis, assesses OSHA recordability, runs investigation interviews via WebSocket, finds precedent matches, generates trend analytics |
| **ER Copilot** | AI parses uploaded documents via RAG, generates investigation timelines, detects discrepancies in witness statements, checks against company policies, suggests outcomes, matches similar cases, generates PDF reports |
| **Pre-Termination** | 9-dimension AI risk scan (ER history, IR involvement, leave/accommodation status, protected activity, documentation quality, tenure/timing, consistency, manager risk, retaliation indicators) with narrative explanation |
| **Risk Assessment** | 5-dimension scoring + Monte Carlo simulations + department heat maps + Gemini-generated executive report with prioritized recommendations |
| **Handbooks** | AI generates jurisdiction-specific handbooks, auto-researches missing topics, detects coverage gaps |
| **Legislative Tracker** | AI scans pending bills using Google Search grounding + prediction markets |

**This is not a content library. It's an AI workforce that does the analysis your HR team and outside counsel currently do manually.**

## Single-Tier Pricing: Full Platform

### Why Single Tier

- **Procurement teams are small** — simpler pricing = faster close
- **Feature gating creates friction** — a 700-employee clinic that can't use ER Copilot after an incident because they're on the wrong tier is a bad experience
- **AI cost is per-company, not per-employee** — Gemini calls scale with activity, not headcount
- **Upsell comes from add-on integrations**, not from withholding core features
